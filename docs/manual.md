## 1. ログイン・ログアウト

### ログイン

1. ブラウザでシステムのURLを開く
2. 「**一般用**」タブが選ばれていることを確認する
3. プルダウンから**自分の名前**を選ぶ
4. 管理者から配布された **6桁のPIN** を入力する
5. 「**ログイン**」ボタンを押す

![ログイン画面イメージ]

> **PINを忘れた場合**：管理者に連絡してPINを再発行してもらってください。

### ログアウト

画面左下のサイドバー最下部にある「**ログアウト**」ボタンを押すと、ログアウトしてログイン画面に戻ります。

## 2. ダッシュボード

ログイン後に最初に表示される画面です。業務の「今」を一目で確認できます。

### 管理者からのお知らせ

画面上部に表示されます。管理者から全スタッフへのお知らせを確認できます。

**管理者へメッセージを送る**

困ったことや連絡事項がある場合は、お知らせパネル右上の「**管理者へメッセージ**」ボタンから送信できます。

1. 「管理者へメッセージ」ボタンを押す
2. 入力者（自分の名前）を確認・入力する
3. 伝えたい内容を入力する
4. 「**送信**」ボタンを押す

### 進行中の修理依頼 / 購入依頼

対応中の依頼がカード形式で一覧表示されます。

- 各カードには現在のステータスとステータスバー（進行度）が表示されます
- 「**次のステータスへ**」ボタンを押すと、担当者名を入力した上でステータスを一歩進められます
- カード右上の「**…**」メニューから詳細の編集・確認ができます

### 定期点検（期限間近）

次回点検日が近い、または期限を過ぎた機器が表示されます。

| 表示色 | 意味 |
|--------|------|
| 赤 | 点検期限が過ぎている |
| 黄 | 今月中に点検期限が来る |

## 3. 修理依頼

修理が必要な機器の依頼を登録・管理する画面です。

### 新規依頼の登録

1. 画面右上の「**新規依頼**」ボタンを押す
2. **修理経路**を選択する

| 修理経路 | 内容 | ステータスの流れ |
|----------|------|----------------|
| **メーカー修理** | 業者に依頼して修理 | 依頼受付 → 確認中 → 選定 → 業者見積依頼 → 見積受取 → 院内決済 → 業者報告 → 修理 → 完了 |
| **自施設修理** | 施設内で修理 | 受付 → 修理中 → 修理完了 → 完了 |

3. 対象機器のバーコードを入力して「**検索**」を押し、機器を選択する
4. 各項目を入力する

| 項目 | 内容 |
|------|------|
| 受付CE | 受付した担当者の名前 |
| 依頼者氏名 | 修理を依頼してきたスタッフの名前 |
| 依頼者部署 | 依頼者の所属部署 |
| 依頼内容 | 不具合の状況や修理してほしい内容 |
| 備考 | その他の補足事項（任意） |

5. 「**登録する**」ボタンを押す

> **自施設修理の場合**、受付時に機器の状態（正常・修理・破棄）を選択します。

### ステータスの進め方

依頼カードの「**次のステータスへ**」ボタンを押すと、次の手順でステータスが進みます。

1. ダイアログが開く
2. **担当者名**を入力する（必須）
3. ステータスによっては追加入力がある
   - 「見積受取」→ 見積金額の入力
   - 自施設修理の「修理完了」→ 修理内容・交換部品の入力
4. 「**確定**」ボタンを押す

### 詳細の確認・編集

カード右上の「**…**」メニュー → 「**詳細・編集**」を開くと、

- 全ステータスへの変更
- 変更履歴の確認
- 依頼の削除

ができます。

## 4. 購入依頼

新しい機器や消耗品の購入依頼を登録・管理する画面です。

**ステータスの流れ**：依頼受付 → 確認中 → 選定 → 業者見積依頼 → 見積受取 → 院内決済 → 業者報告 → 購入 → 完了

### 新規依頼の登録

1. 画面右上の「**新規依頼**」ボタンを押す
2. 購入希望の機器名を入力する（バーコード検索は不要）
3. 受付CE・依頼者氏名・部署・依頼内容を入力する
4. 「**登録する**」ボタンを押す

ステータスの進め方・詳細の確認は「[3. 修理依頼](#3-修理依頼)」と同様です。

## 5. 機器台帳

院内にある医療機器の一覧を管理する画面です。

### 機器の検索・絞り込み

画面上部の検索欄に以下のいずれかを入力すると絞り込めます。

- ME No.（バーコード番号）
- 機種名
- メーカー名
- 型式

さらに、ステータスでのフィルター（利用中・修理中・移動・破棄）も使えます。

### 機器カルテの確認

機器の行をクリックすると、画面右側に「**機器カルテ**」が表示されます。

| 情報 | 内容 |
|------|------|
| 基本情報 | ME No.・機種名・メーカー・型式・購入日・次回点検日など |
| 修理履歴 | これまでの修理依頼の一覧 |
| 点検履歴 | これまでの点検記録の一覧 |

### 機器の登録

1. 「**新規登録**」ボタンを押す
2. 以下の項目を入力する

| 項目 | 内容 |
|------|------|
| ME No.（バーコード） | 機器に貼付されているバーコード番号（必須） |
| 機種名 | 機器の名称（必須） |
| メーカー | 製造メーカー名 |
| 型式 | 機器の型式・モデル番号 |
| 購入日 | 購入した日付 |
| ステータス | 利用中 / 修理中 / 移動 / 破棄 / 不明 |
| 次回点検予定日 | 次に点検が必要な日付 |
| 備考 | 設置場所や注意事項など |

3. 「**登録する**」ボタンを押す

> **複製登録**：同じ機種の別台番を登録するときは「**…**」メニュー → 「**複製して新規登録**」を使うと、情報を引き継いで素早く登録できます。

### 機器の編集

1. 機器の行の「**…**」メニュー → 「**編集**」
2. 内容を変更して「**保存する**」

## 6. メンテナンスマスタ

サイドバーの「**メンテナンスマスタ**」から開きます。機種（メーカー・型式）ごとに点検チェックリストや点検サイクルを登録する画面です。

登録した内容は、定期点検・日常点検の記録画面に自動で反映されます。

### 画面の構成

画面上部に **2つのタブ** があります。

| タブ | 用途 | 反映先 |
|------|------|--------|
| **定期点検** | 年次・月次などの定期点検用 | 定期点検画面・ダッシュボード・年間メンテ計画 |
| **日常点検** | 毎日のルーティン点検用 | 日常点検画面のみ |

### 一括テンプレートの作成

複数の型式で共通して使う点検項目を、あらかじめテンプレートとして登録できます。

1. 「**テンプレートを選択**」で「**＋ 新規テンプレート**」を選ぶ
2. **マスタ名（テンプレート名）** を入力する（例：シリンジポンプ共通点検）
3. 点検項目を追加する（下記「点検項目の種類」参照）
4. 「**テンプレートを保存**」ボタンを押す

既存のテンプレートはプルダウンから選んで内容の確認・編集・削除ができます。

### メーカー・型式マスタの登録

機器台帳に登録されているメーカー・型式ごとに、点検内容を設定します。

#### 手順

1. タブ（定期点検 / 日常点検）を選ぶ
2. 「**台帳・マスタから型式を選択**」で対象のメーカー・型式を選ぶ  
   （一覧にない場合は「手入力でメーカー・型式を指定…」から直接入力）
3. 必要に応じて各項目を入力する

| 項目 | 内容 |
|------|------|
| 製造元メーカー | 機器のメーカー名（例：TERUMO） |
| 型式 | 機器の型式（例：TE-332S） |
| メンテナンス方法 | 点検手順・清掃方法・注意事項など（点検実施時に参照できます） |
| 点検期間（定期点検のみ） | 1ヶ月・3ヶ月・6ヶ月・12ヶ月など、定期点検のサイクル |

4. **点検項目**を追加する（テンプレートから読み込んでも可）
5. 「**型式マスタを保存**」ボタンを押す

#### テンプレートから点検項目を読み込む

1. 「**一括テンプレートから読み込む**」でテンプレートを選ぶ
2. 「**項目を適用**」ボタンを押す
3. テンプレートの点検項目が型式マスタにコピーされる

> すでに項目がある場合は、テンプレートの内容で置き換えるか確認ダイアログが表示されます。

#### 点検項目の種類

「項目を追加」メニューから、次の種類を選んで登録できます。

| 種類 | 点検時の入力形式 |
|------|----------------|
| チェックボックス | チェックの有無 |
| 数値＋単位 | 数値と単位（例：電圧 V） |
| Y/N | はい / いいえ |
| 日付 | 日付の入力 |
| 自由記入 | テキスト入力 |
| 備考 | 補足メモ |
| 点検者 | 点検担当者名 |
| 適／不適／対象外 | ○ / × / — の3択（旧形式） |

日常点検マスタでは、項目ごとに実施頻度（毎日など）も設定できます。

### 初回の点検計画（月均等）— 定期点検のみ

新しい型式マスタを登録した直後は、対象機器がダッシュボードで「未実施」と表示されることがあります。

「**初期計画を組む（月均等）**」ボタンを使うと、同じ型式の機器に次回点検予定日を自動で割り当てられます。

- 対象：定期点検の記録がなく、次回点検予定も未設定の機器
- 割り当て方法：当年の1月〜12月に均等配分（各月15日）
- マスタを新規保存した直後に、確認ダイアログから実行することもできます

2回目以降の点検予定日は、点検記録を登録すると自動で更新されます。

### 型式マスタの削除

「**型式マスタを削除**」ボタンでマスタを削除できます。

- 既存の点検記録は残ります（マスタとの紐づけのみ解除）
- 機器台帳の次回点検予定日は変更されません

### よくある使い方の流れ

```
① 一括テンプレートを作成（共通の点検項目）
    ↓
② メーカー・型式マスタにテンプレートを適用
    ↓
③ 型式マスタを保存
    ↓
④（定期点検のみ）初期計画を組む
    ↓
⑤ 定期点検 / 日常点検画面で点検を記録
```

> **ポイント**：機器台帳にメーカー・型式が登録されていないと、マスタの型式一覧に表示されません。先に機器台帳でメーカー・型式を入力してください。

## 7. 定期点検

機器の定期点検を記録する画面です。

### 点検の記録手順

1. バーコードリーダーで機器のバーコードをスキャン、または ME No. を手入力して「**検索**」を押す
2. 機器が表示されたら内容を確認する
3. **実施日**を入力する（デフォルトは今日の日付）
4. チェックリストの各項目を選択する

| マーク | 意味 |
|--------|------|
| ○ | 正常 |
| × | 異常あり |
| — | 対象外・未実施 |

5. 異常があった場合は**備考**に詳細を記入する
6. 「**記録する**」ボタンを押す

> 点検を記録すると、次回点検予定日が自動的に更新されます。チェックリストの内容は「[6. メンテナンスマスタ](#6-メンテナンスマスタ)」で登録した型式マスタに基づいて表示されます。

## 8. 日常点検

サイドバーの「**日常点検**」から開きます。毎日行うルーティン点検の確認・記録を行う画面です。

### 本日の日常点検一覧

画面上部に、**本日実施すべき日常点検**の一覧が表示されます。

- **未実施** / **完了** の状態がバッジで示されます
- 「**点検へ**」ボタンから、その機器の点検記録画面に進めます
- 一覧はリアルタイムで更新されます（「更新」ボタンでも再読み込み可能）

### 点検の記録手順

1. 一覧から機器を選ぶか、バーコードをスキャンして機器を読み込む
2. 日常点検専用のチェックリストに従って各項目を記録する
3. 「**記録する**」ボタンを押す

チェックリストの内容は、メンテナンスマスタの「日常点検」タブで登録した内容が使われます。

## 9. 年間メンテ計画

機器ごとの年間点検スケジュールを確認できる画面です。

- 月ごとに点検が必要な機器の一覧が表示されます
- 実施済み・未実施・期限超過の状況が色で示されます

点検記録の登録は「[7. 定期点検](#7-定期点検)」から行ってください。

---

## 10. 履歴管理

完了済みの依頼や実施済みの点検記録を検索・確認する画面です。

### 修理依頼 / 購入依頼の履歴

- キーワード・日付・担当者などで絞り込みができます
- 依頼行をクリックすると詳細（変更履歴・見積金額など）を確認できます
- 必要に応じて内容の編集や削除もできます

### 点検記録の履歴

- 実施日・機器名などで絞り込みができます
- 誤って登録した記録は行右端の「**削除**」ボタンで削除できます

## よくある質問

**Q. バーコードスキャンができない**
バーコードリーダーを検索欄にフォーカスした状態でスキャンしてください。入力後に「検索」ボタンを押すか、Enterキーを押します。

**Q. ログインできない**
PINが正しいか確認してください。それでもログインできない場合は管理者にお問い合わせください。

**Q. 依頼のステータスを間違えて進めてしまった**
依頼カードの「…」メニュー → 「詳細・編集」を開くと、ステータスを任意の値に変更できます。

**Q. 点検期限が過ぎている機器がダッシュボードに表示されている**
定期点検画面から点検記録を登録すると、次回点検予定日が更新されてダッシュボードの表示が消えます。

**Q. 点検画面にチェックリストが表示されない**
メンテナンスマスタで、対象機器のメーカー・型式に一致するマスタが登録されているか確認してください。機器台帳のメーカー・型式の表記がマスタと一致している必要があります。
